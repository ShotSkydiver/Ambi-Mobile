## Environment Configuration

To get started setting up the project and running it locally:

1. Install the dependencies by running `yarn install`.

2. Create a file called `google-services.json` in the directory `android/app/` (get the contents of file from vault).

3. Create a file called `GoogleService-Info.plist` in the directory `ios/AmbiNext/` (get the contents of file from vault).

4. Run the app on either platform by executing either `react-native run-ios` or `react-native run-android`.

Steps 2 and 3 are required for the push notifications to work.

> If you want to generate and install the apk on your android device or an emulator,
> then please follow the steps here => https://facebook.github.io/react-native/docs/signed-apk-android
